A fast Discord message sender that reads lines from a file and sends them one by one to a target channel.

## Requirements

- Python 3.7+
- `requests`
- `pyfiglet`

Install dependencies:

```bash
pip install requests pyfiglet
```

## Usage

```bash
python anthrax.py
```
